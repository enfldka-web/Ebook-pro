## Atlas v0.2 — 2026-07-20

- **상태**: 완료
- **핵심 업데이트**: 단일 `index.html` 파일 구조를 `css/`·`js/` 파일로 분리(프론트엔드 구조 모듈화)
- **사용자에게 달라진 점**: 없음 — 화면, 기능, 조작 방식은 이전과 완전히 동일합니다. 이번 버전은 유지보수
  기반을 다지는 내부 개선입니다.
- **알려진 제한사항**: 없음(코드 재배치만 수행, 기존 동작 유지)
- **검증 내용**: 분리 전후 코드 내용이 diff 상 동일함을 확인, Playwright로 로그인·전자책 생성·저장·API
  키·로그아웃 등 핵심 기능 회귀 테스트 통과
- **관련 PR**: #1

## Atlas v0.3 — 2026-07-21

- **상태**: 완료
- **핵심 업데이트**: 결과 화면에 "🎯 Thumbnail Studio" 진입 버튼 추가. 후킹 문구 선택, 텍스트 편집,
  템플릿 8종, 색상 6종, 레이아웃 6종, 이미지 스타일 6종을 고르며 652×488 실시간 미리보기로 확인 가능.
  이미지 생성용 Prompt 자동 생성/복사/수정, PNG·JPG 다운로드, 프로젝트 저장/불러오기 지원.
- **사용자에게 달라진 점**:
  - 새 화면(Thumbnail Studio)에서 썸네일을 직접 선택·수정·미리보기하며 만들 수 있게 됨
  - 기존 "🛒 썸네일 + 상세페이지" 화면은 그대로 남아 있으며, 새 화면과 별개로 계속 사용 가능
- **알려진 제한사항**:
  - Hook 추천 문구는 고정된 fallback 목록에서 나오며, 실제 AI가 매번 새로 만들어주지는 않음
  - 이미지 스타일(Flat/Photo/3D 등)은 실제 이미지를 생성하지 않고, 다운로드용 Prompt 문구에만 반영됨
- **검증 내용**: Playwright로 신규 기능 22개 항목 + 기존 기능 회귀 4개 항목 확인, PNG/JPG 다운로드 파일이
  실제로 652×488 픽셀임을 직접 확인
- **관련 PR**: #2

## Atlas v0.4 — 2026-07-21

- **상태**: 완료
- **핵심 업데이트**: Thumbnail Studio에 "Thumbnail Score" 분석 패널 추가. 총점과 8개 항목별 점수, 개선
  제안, Hook·색상·레이아웃·이미지 스타일 추천을 확인하고 "적용" 버튼으로 바로 반영 가능.
- **사용자에게 달라진 점**:
  - 미리보기 아래에서 현재 썸네일의 내부 품질 점수를 확인할 수 있음(총점은 항상 보이고, 상세 내용은
    펼쳐서 볼 수 있음)
  - 점수가 낮은 부분에 대한 구체적인 개선 제안을 확인할 수 있음
  - 추천 Hook/색상/레이아웃/스타일을 클릭 한 번으로 바로 적용할 수 있음
- **알려진 제한사항**:
  - **이 점수는 실제 클릭률·판매량·매출을 예측하거나 보장하지 않습니다.** 정해진 기준에 따라 계산한
    내부 참고 점수입니다.
  - 추천은 실제 AI 분석이 아니라 정해진 규칙(제목·카테고리 키워드 등) 기반입니다.
- **검증 내용**: Playwright로 점수 재현성(같은 입력 → 같은 점수), 항목별 점수 변화, 추천 적용 시 미리보기·
  점수 자동 갱신, 기존 기능 회귀까지 29개 항목 확인
- **관련 PR**: #3

## Atlas v0.5 — 2026-07-21

- **상태**: 완료(병합 대기 — 403으로 직접 push 불가, ZIP을 통한 반영 대기)
- **핵심 업데이트**: 결과 화면에 "📄 Sales Page Studio" 진입 버튼 추가. 상세페이지를 8개 섹션(히어로·문제
  공감·해결책·목차·핵심 장점·비포/애프터·추천 대상·마무리 CTA)으로 구성해 On/Off·순서 변경·문구 편집이
  가능하고, 색상 테마 6종·카드 레이아웃 6종을 선택하며 실시간 미리보기로 확인할 수 있습니다. 개별 카드
  PNG 다운로드와 전체 카드 ZIP 일괄 다운로드를 지원합니다.
- **사용자에게 달라진 점**:
  - 새 화면(Sales Page Studio)에서 상세페이지 섹션을 켜고 끄거나 순서를 바꾸며 원하는 구성으로 만들 수 있게 됨
  - 기존 "🛒 썸네일 + 상세페이지" 화면(9장 카드뉴스)은 그대로 남아 있으며, 새 화면과 별개로 계속 사용 가능
  - 전체 상세페이지를 ZIP 파일 하나로 한 번에 내려받을 수 있게 됨(기존 화면에는 없던 기능)
- **알려진 제한사항**:
  - FAQ 섹션은 이번 버전에 포함되지 않음(후속 버전 후보)
  - 섹션 순서 변경은 위/아래 버튼만 지원하며 드래그 앤 드롭은 지원하지 않음
  - 상세페이지 품질 점수/추천(Sales Page Intelligence)은 포함되지 않음
  - 문구는 실제 AI가 새로 만들어주는 것이 아니라 기존 전자책 데이터를 규칙 기반으로 채워 넣은 뒤 사용자가 직접 수정하는 방식
- **검증 내용**: Playwright로 신규 기능(섹션 구조·편집·테마·레이아웃·Live Preview·Export·저장불러오기) +
  기존 기능(9장 카드뉴스, Thumbnail Studio, Thumbnail Intelligence) 회귀까지 확인, 개별 PNG 다운로드
  파일이 실제로 1080×1350 픽셀임을 직접 확인
- **관련 PR**: 병합 후 갱신 예정
